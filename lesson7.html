Адаптивність у кодуванні

Адаптивність (англ. responsive design) — це підхід до створення вебсайтів, які гарно виглядають і зручно працюють на будь-яких пристроях: від великих моніторів до смартфонів.

Що таке адаптивність?

Адаптивність означає, що вебсторінка "підлаштовується" під розмір екрана користувача, не змінюючи зручність користування чи візуальну якість. Це досягається за допомогою:

  • гнучких сіток (flex, grid)
  • відсоткових розмірів
  • медіа-запитів (media queries)
  • відповідних шрифтів та зображень

Основні інструменти адаптивного дизайну

1. Відсотки й vw / vh


                  .container {
                        width: 100%;
                        padding: 5vw;
                  }

2. Flexbox та Grid Layout

.
                  container {
                        display: flex;
                        flex-wrap: wrap;
                        gap: 20px;
                  }

3. Media Queries (медіа-запити)


                  @media (max-width: 768px) {
                        .container {
                              flex-direction: column;
                        }
                  }

4. Viewport meta tag


                  <meta name="viewport" content="width=device-width, initial-scale=1.0">

Приклад адаптивного блоку


                  <div class="card-container">
                  <div class="card">Блок 1</div>
                  <div class="card">Блок 2</div>
                  <div class="card">Блок 3</div>
                  </div>

Де ще застосовується адаптивність?

  • Навігаційні меню, які перетворюються на бургер-меню
  • Зображення, які змінюють розмір або формат
  • Типографіка, що змінює розмір залежно від екрану
