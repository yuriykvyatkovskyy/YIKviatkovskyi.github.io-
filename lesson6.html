Урок 6: Flexbox

Тривалість: 45 хв

Дивитися відеоДомашнє завдання

Відео


Flexbox

Теорія

Flexbox CSS

Flexbox (Flexible Box Layout) — це система розкладки, яка дозволяє розміщувати, вирівнювати та розтягувати елементи в одному напрямку: по горизонталі або вертикалі.

Основна структура


                  <div class="container">
                  <div class="item">1</div>
                  <div class="item">2</div>
                  <div class="item">3</div>
                  </div>

                  .container {
                        display: flex;
                  }

Властивості для контейнера

flex-direction напрямок головної осі: row, column
justify-content вирівнювання по горизонталі
align-items вирівнювання по вертикалі
gap відстань між елементами

flex-direction


                  .container {
                        flex-direction: row; /* або column */
                  }

justify-content

Значення Пояснення
flex-start зліва
flex-end справа
center по центру
space-between відстань між елементами
space-around з відступами навколо
space-evenly рівномірні відступи

align-items

Значення Пояснення
stretch розтягнути по висоті (за замовчуванням)
flex-start вирівняти до верху
flex-end вирівняти до низу
center по центру

Властивості для елементів усередині

Властивість Опис
flex-grow як сильно елемент може розширюватись
flex-shrink як сильно може стискатись
flex-basis початковий розмір
flex скорочення: flex: grow shrink basis
align-self перевизначає align-items для окремого елемента 

                  .item {
                        flex: 1;
                  }

                  .item:first-child {
                        flex: 2;
                  }
