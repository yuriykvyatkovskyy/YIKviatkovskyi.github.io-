Основи CSS

Теорія

CSS (Cascading Style Sheets) — це мова стилів, що використовується для опису зовнішнього вигляду веб-сторінок, написаних мовою розмітки, як-от HTML. Вона визначає, як елементи веб-сторінки мають виглядати на екрані, папері чи інших носіях, зокрема, колір, розмір шрифту, розташування та інші візуальні аспекти.

  • Розділення структури та стилю: CSS дозволяє відокремити зміст (HTML) від його оформлення, що робить код чистішим і дозволяє керувати виглядом багатьох сторінок одночасно.
  • Мова стилів: Це не мова програмування, а набір правил, які описують, як повинен виглядати певний елемент веб-сторінки.
  • Застосування: CSS використовується для стилізації елементів HTML, як-от заголовки, параграфи, таблиці, посилання, а також для створення складних макетів сторінок за допомогою таких технологій, як CSS Grid або Flexbox.
  • Специфікація: CSS стандартизовано Консорціумом Всесвітньої павутини (W3C) і поділено на модулі, які розробляються та випускаються окремо.

Для чого потрібен CSS:

  • Стилізація: Можна змінювати кольори, шрифти, розмір тексту, фон та інші візуальні аспекти елементів.
  • Макетування: Дозволяє керувати розташуванням елементів на сторінці, створювати сітки та багатоколонкові макети.
  • Адаптивність: Можна налаштовувати різні стилі для різних пристроїв (комп'ютерів, планшетів, телефонів) і розмірів екрана.
  • Анімація та ефекти: Дає змогу додавати анімації, ефекти наведення курсору та інші декоративні елементи.

HTML та CSS у парі
HTML створює структуру вебсторінки (фундамент), а CSS додає стиль та естетику (оформлення будинку). Їхнє спільне використання дозволяє розділити вміст від візуального оформлення, що полегшує підтримку та зміну дизайну сайту

Як працює "каскадність"
Слово "каскадні" у назві означає, що стилі, визначені на різних рівнях, можуть впливати на елементи. Якщо для одного елемента існує кілька правил, CSS визначає, яке правило має вищий пріоритет. Наприклад, стиль, застосований до батьківського елемента, успадковується його дочірніми елементами, якщо не визначено інше.

Кольори в CSS
У CSS кольори можна задавати кількома способами:

  • Назвою кольору: color: red;
  • Шістнадцятковим кодом: color: #ff0000;
  • RGB: color: rgb(255, 0, 0);
  • RGBA (з прозорістю): color: rgba(255, 0, 0, 0.5);
  • HSL: color: hsl(0, 100%, 50%);
    Основні властивості:
  • color — колір тексту
  • background-color — фоновий колір

Шрифти в CSS
CSS дозволяє налаштовувати вигляд тексту:

    Основні властивості:
  • font-family — тип шрифту (наприклад: font-family: Arial, sans-serif;)
  • font-size — розмір шрифту (font-size: 16px;)
  • font-weight — товщина шрифту (bold, normal, lighter, 700)
  • font-style — стиль (italic, normal)
  • text-align — вирівнювання (left, center, right)
  • line-height — міжрядковий інтервал

Рамки в CSS
Рамки (borders) додають контур навколо елементів:

    Основні властивості:
  • border-width — товщина рамки
  • border-style — тип лінії (solid, dashed, dotted, none)
  • border-color — колір рамки

    • Скорочено - border: 2px solid black;
    Можна задати рамки окремо:
    border-top, border-right, border-bottom, border-left

Приклад CSS-коду:
p {
color: navy;
font-family: "Times New Roman", serif;
font-size: 18px;
border: 1px dashed gray;
background-color: #f0f0f0;
}
