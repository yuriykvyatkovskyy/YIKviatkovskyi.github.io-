Теорія

Блок-модель (Box Model) у CSS
Уяви собі HTML-елемент як коробку. Усі елементи на сторінці, незалежно від того, текст це чи кнопка, — мають свою коробку. Ця блок-модель складається з чотирьох основних шарів:

  1. margin — зовнішній відступ

    2. Це простір зовні елемента. Він відділяє елемент від інших на сторінці.
    Приклад:
    div {
    margin: 20px; /* з усіх боків */
    }
  2. padding — внутрішній відступ

    3. Це простір всередині елемента, між вмістом і рамкою (border).
    Приклад:
    div {
    padding: 15px; /* текст буде "дихати", не прилипати до країв */
    }
  3. border — рамка навколо контенту й падінгу

    4. Це лінія навколо елемента. Її можна стилізувати: товщина, колір, тип (суцільна, пунктирна тощо).
    Приклад:
    div {
    border: 2px solid black;
    }
  4. display — спосіб відображення елемента

    5. Це властивість визначає як елемент поводиться у потоці документа.
      Найважливіші значення:
    • block - Блочний елемент: займає весь ряд (наприклад, <div> <p>)
    • inline - Рядковий елемент: займає лише ширину свого вмісту (наприклад, <span>)
    • inline-block - Комбінація: поводиться як inline, але можна задавати width, height, margin, padding
    • none - Елемент взагалі не відображається
    • flex / grid - Сучасні системи розкладки (пояснення на окрему тему)

    Створимо "коробку", яка:
  • матиме 10px простору навколо себе (margin),
  • 20px простору всередині перед текстом (padding),
  • рамку товщиною 1px,
  • поводитиметься як блок (на новому рядку).

.box {
margin: 10px;
padding: 20px;
border: 1px solid gray;
display: block;
}
